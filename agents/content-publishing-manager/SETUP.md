# Set up Content Publishing Manager

This is the standalone Eve agent. It does not require a channel, connector, or external credential.

1. Keep every downloaded path intact.
2. Place the directory where you run Eve agents.
3. Review `instructions.md` and the skill file before use.
4. Run the agent from this directory:

```bash
npx eve@latest
```

5. Use `examples/sample-input.md` for a first verification run.

To connect a service or conversation surface, download one of this agent's integration variants and follow its replacement `SETUP.md` and `.env.example`.
